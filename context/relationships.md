# Relationships

Key people to stay in contact with. HEARTBEAT flags contacts going cold based on their cadence.

**Cadence guide:**
- `weekly` — team, close collaborators (flag if >7 days)
- `monthly` — investors, mentors, warm leads (flag if >30 days)
- `quarterly` — advisors, network nodes, past colleagues worth keeping warm (flag if >90 days)
